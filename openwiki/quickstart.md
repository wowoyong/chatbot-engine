## OpenWiki Quickstart

Welcome to the OpenWiki documentation for the chatbot-engine repository. This wiki provides an overview of the project's architecture, key components, and how to interact with the system.

### Getting Started
1. **Explore the Architecture**: Understand the system's design and major components.
2. **Core Engine**: Learn about the phases of development and core functionalities.
3. **Memory Management**: Discover how session management and context handling are implemented.
4. **RAG Integration**: Explore the Retrieval-Augmented Generation features and their implementation details.
5. **CLI Usage**: Get started with the command-line interface for interacting with the chatbot.
6. **Testing and CI/CD**: Review the testing strategies and GitHub Actions workflows.
