## CLI Commands

The chatbot-engine provides a command-line interface (CLI) for interacting with the system. Key commands include:

### Installation
- **npm install**: Installs the necessary dependencies for the chatbot-engine.

### Running the Chatbot
- **npm start**: Starts the chatbot application.
- **node src/cli/main.ts**: Runs the CLI interface for user interaction.

### Testing
- **npm test**: Executes the test suite using Vitest.
- **vitest**: Runs individual tests or specific test files.

### Documentation Updates
- **GitHub Actions**: Automates the updating of OpenWiki documentation through workflows defined in .github/workflows/openwiki-update.yml.

### Key Files
- **src/cli/main.ts**: Main entry point for the CLI interface.
- **.github/workflows/openwiki-update.yml**: Defines the workflow for automated documentation updates.
- **package.json**: Contains configuration for npm scripts and dependencies.

These CLI commands facilitate development, testing, and maintenance of the chatbot-engine.
